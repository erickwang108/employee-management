# Introduction
Simple employee management build with electron and react.
# Installation

`yarn or npm install`
# Run in dev mood

`yarn dev` or `npm run dev`

# Upgrade dev tools

If you have an error like 'state is undefined', please use commend:
`UPGRADE_EXTENSIONS=1 yarn dev` to upgrade your dev tools.
